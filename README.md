# BizTask - ビジネスタスク管理アプリ

スマホで使えるビジネス向けタスク管理PWAアプリです。

## 機能

- タスクの追加・編集・削除・完了管理
- 優先度設定（高・中・低）
- カテゴリ分類（業務・会議・連絡・企画・経理・その他）
- 期限設定と期限切れアラート
- フィルタリング（すべて・未完了・緊急・今日・完了）
- ソート（作成日順・期限順・優先度順）
- レポート画面（達成率・優先度別・カテゴリ別統計）
- データはローカルストレージに保存
- PWA対応（ホーム画面に追加可能）

## 使い方

`index.html` をブラウザで開くだけで使えます。
スマホのブラウザで「ホーム画面に追加」するとアプリのように使えます。
